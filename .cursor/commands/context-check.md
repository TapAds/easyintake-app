---
name: context-check
description: Confirm Cursor knows what project it is working on
---

Without reading any files, answer these questions 
from your current context window only:

1. What project are you working on right now?
2. What is the tech stack?
3. What does this project own vs what does it 
   delegate to external systems?
4. What authentication approach is in use, if any?
5. What are the three most important things 
   you must NOT do in this codebase?

If you are uncertain about any answer, say so — 
do not guess. Uncertainty means the context 
documents need updating.

After answering, read the primary project context document(s)
(use this repository’s canonical names from README or docs index—e.g. CONTEXT.md)
and confirm whether your answers were accurate.
